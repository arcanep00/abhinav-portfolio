# Abhinav Pandey Portfolio

Premium developer portfolio built with Next.js, TypeScript, Tailwind CSS, and Framer Motion.

## Focus Areas

- Software Developer
- Backend Developer
- Django Developer
- Python Developer
- Odoo Developer

## Highlighted Stack

- Python
- Django
- Odoo ERP
- PostgreSQL
- REST APIs
- Git/GitHub
- Linux

## Run Locally

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.

## Production Build

```bash
npm run build
npm run start
```
